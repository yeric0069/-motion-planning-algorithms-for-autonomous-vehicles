# -motion-planning-algorithms-for-autonomous-vehicles
There are two sets of tasks : 1) Sampled-based planning algorithms  2) Optimisation-based planning.
The main task is to perform local motion planning for an autonomous vehicle 
(one at the left-bottom corner) to complete a valet parking, as shown in Fig.1, which is drawn by 
parkinglot.m (download it from Moodle). The vehicles in the parking lot are assumed identical and 
the dimension of the vehicle is shown in Fig. 2. The initial position of the vehicle is (0.2,3.0) and 
the goal position is (13.2,16.0). Note that the autonomous vehicle’s position cannot be negative 
(i.e. it is bounded by x ≥ 0 and y ≥ 0) due to the parking lot boundary.
